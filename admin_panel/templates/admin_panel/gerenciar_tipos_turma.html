Como Usar o Sistema de Tipos de Turma

  1. Criar Competências: Primeiro, certifique-se de ter criado as competências necessárias na seção Gerenciar Competências.
  2. Criar Tipos de Turma: Defina os tipos de turma (Ex: Basic 1, Basic 2, High Resolution 4) e associe as competências apropriadas.
  3. Criar Turmas: No admin do Django, ao criar uma turma, selecione o tipo de turma e defina apenas o identificador (Ex: TT18, MW20).
  4. Competências Automáticas: As competências serão automaticamente herdadas do tipo de turma selecionado.
Exemplos de Identificadores:
  • TT18 - Tuesday/Thursday 18h
  • MW20 - Monday/Wednesday 20h
  • SEG19 - Segunda-feira 19h
  • SABMANHA - Sábado de manhã
